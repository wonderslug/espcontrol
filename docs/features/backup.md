# Backup

You can save your entire panel configuration as a file and restore it later. You'll find these options in the **Settings** tab on the [Setup](/features/setup) page, under the **Backup** section.

- **Export** - saves your entire setup (buttons, subpages, colours, brightness, screen schedule, and display settings) as a file you can keep as a backup.
- **Import** - loads a previously saved file to restore your setup. If you're loading a backup from a different-sized panel, the buttons are rearranged to fit automatically.
